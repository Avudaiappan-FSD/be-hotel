# Hotel Management System

## Description

This is a simple hotel management system that allows users to manage hotel bookings, rooms, and customer information. The system provides functionalities for adding new bookings, checking room availability, and managing customer details.

## Features
- Add new bookings
- Check room availability
- Manage customer information
- View booking history
- Generate invoices
- Manage room types and rates
- Track occupancy rates
- Handle check-in and check-out processes
- Send booking confirmations

Hotel Booking Persons:

- Book rooms
- Cancel bookings
- View booking history
- Update personal information
- Receive booking confirmations
- Provide feedback on their stay
- View available room types and rates
- register for loyalty programs

Receptionist:

- Manage bookings and reservations
- Check room availability
- Handle check-in and check-out processes
- Update customer information
- Generate invoices and receipts
- Manage room types and rates
- Track occupancy rates
- Handle customer complaints and feedback
- Provide assistance to guests during their stay
- Coordinate with housekeeping and maintenance staff
- Ensure smooth operation of the hotel front desk

administrator:

- Manage user accounts and permissions
- Configure system settings and preferences
- Monitor system performance and security
- Generate reports and analytics
- Manage hotel policies and procedures

## Endpoints

-Post / Users/register: Register a new user
-Post / Users/login: Login a user
-Get / Users/logout: Logout a user
-Get / Users/profile: Get user profile information
